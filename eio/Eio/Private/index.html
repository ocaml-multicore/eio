val default_traceln : ?__POS__:(string * int * int * int) -> ( 'a, Stdlib.Format.formatter, unit, unit ) Stdlib.format4 -> 'a

default_traceln is a suitable default implementation for Eio.Std.traceln.

It writes output to stderr, prefixing each line with a "+". If __POS__ is given, it also displays the file and line number from that. It uses mutex so that only one domain's output is written at a time.
